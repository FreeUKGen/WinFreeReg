WinFreeREG - File Workspace

The File Workspace opens after the user has double-clicked on a file in the Local Files list. it shows a scrollable grid of all the records on the file and allows the user to edit the fields and add and remove records.

Outwardly, the grid works in much the same way as it did in WinREG - although some of that functionality (such as duplicating data from a previous record) has not been included in this program.

There is a button on the navigation bar that when pressed, allows the user to change some of the details kept in the header records of the transcription file. See File Details for more details.

Extras

Right-clicking anywhere on the record display area results in the appearance of a context menu. At the moment, the available options are limited, allowing access to the FreeREG Tables and, for files of Burials and Marriages, access to the User Tables.
