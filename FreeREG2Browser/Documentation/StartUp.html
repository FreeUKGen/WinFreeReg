WinFreeREG - Starting Up

This program is the intended replacement for the original WinREG. As with the previous incarnation, it provides a transcribing tool for the creation of files that can be uploaded to the FreeREG web site. Unlike the previous version, it provides transcribers with an integrated means of uploading and managing those files on the new web site.

When the program first starts, this window will require the user to fill in some fields: principally the user details and some preferences. Once these have been completed, the Begin button can be pressed to show the main working area - the FreeREG Portal. Upon subsequent startups, this form will continue to be shown filled-in with the settings as they were when the program was shutdown. This allows you to work with multiple FreeREG User IDs and perhaps make use of multiple locations for storing transcription files - different locations for different User IDs.

First Steps

On the first opening of the program, fill in your Name and your email address. This email address can be different to the email address that FreeREG currently associates with you.

Next, fill in your FreeREG User ID and password. These are what identify you to the FreeREG application. Your password is hidden by default, but can be made visible by clicking on the small, underlines link next to the field. A second click on the link will hide a password that is already showing.

The next item to configure is that of your Default County. This is the county that you normally transcribe for. It is used when starting to create a new transcription file, since it controls which places you can transcribe for and, more importantly, the spelling of those place and church names.

The Url is the address of the FreeREG server and should, under normal circumstances, not need changing. The only time this will/might need changing is when this piece of software and the associated server routines are made live. At that time, all new installations of the program will default to the live FreeREG server. Existing installations can either be removed and the reinstalled, or the Url changed manually.

Finally, the Transcription File location is the path to where your transcription files are kept. By default, this will be set to the default path as used by the previous version of WinREG. If this isn't where the files are stored, then you are quite at liberty to use the small Browse link to select the correct place.

After the initial login, you should definitely download some data from the FreeREG server. You need to populate some tables with data held on the server and which will make the uploading of future transcription files less prone to errors. See FreeREG Tables

Subsequent Openings

After the initial set up of the starting environment, the fields can be left alone. just click on the Begin button to proceed. Alternatively, for those users (county coordinators?) that use multiple User IDs, change the FreeREG User ID and password to login under a different identity. If required, you can change the location of the Transcription Files if you maintain files for two or more different User IDs. For now, the program will only remember the last set of startup parameters that it is given. A future enhancement might permit multiple profiles to be stored and one of them selected when the program starts up.

The small, Keep open checkbox next to the Begin button, allows the user to either redisplay the Startup window when returning from the main portal window, or to have it close automatically.
