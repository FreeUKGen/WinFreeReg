Missing File Details

You have arrived at this page because a crucial element of an existing transcription file is incorrect. Generally, this means either the Placename or Churchname in the existing file, does not appear with the given spelling in FreeREG. This will happen because the file was created before WinFreeREG was available to provide the correct spellings for these vital elements.

The elements that are incorrect are indicated in red. Correction of the problem is a simple process. simply ensure that the correct place and church names are selected in the drop-down boxes and press the OK button.
