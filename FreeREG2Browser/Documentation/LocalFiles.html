Local Files List

The Local Files List does is what it is: it's a list of all the transcription files that are found in the default folder or in the Transcriptions Library. What it tries to be is the central point through which transcription files can be edited, created and uploaded to the FreeREG server.

For ease of use, the text of records will be colour-coded. Files in black need nothing doing to them. They haven't been changed since they were last uploaded. Files in green have been changed and most likely need replacing on the server. And files in red are new files and need to be uploaded to the server.

Initially all local files will be in red unless you have first logged on to the server and downloaded a list of the files that have previously been uploaded to the server. This causes the date uploaded to appear and from that, and the date last changed, the program can decide what category each file falls into.

Sorting

The list is normally displayed in alphabetic sequence of filename. The only other column by which the list can be sorted is the Date Uploaded. This is achieved by double-clicking on the column header. Alternatively, right-clicking the Name or Date Uploaded column headings will bring up a context menu, in which the clicked column can be sorted in ascending or descending sequence.

There is also an Unsort option which will return the list to the original sequence in which it was loaded.

Grouping

When the list is showing the contents of the Library,the files will, if there if more than one folder in the library, group the files according to which folder they reside in.

The Date Uploaded column has the extra capability of being able to group records together. Thus, all new files become a group as do all changed and unchanged files. Grouping is available on the context menu shown when the Data Uploaded column heading is right-clicked. When the list is grouped like this, separator lines will appear denoting the category of each group. On the extreme right-hand end of these lines will an arrow. Initially the arrows will poit upwards showing that the group is expanded. Clicking the arrow makes it point downwards and condenses the group. The context menu also has an option to turn the groupings off.

File Summary

Hovering the mouse pointer over a file name and holding the Ctrl key down at the same time results in the program displaying a summary box containing some information about that file. This includes in Place and Church names and the number of records currently in the file.

Starting A New File

Creating a new file is simply a matter of clicking on the Start New File button underneath the file list. See Starting a new file for more details.

Editing A File

To start editing an existing file, simply double-click anywhere on the line in the list. As an aid, the selected line will have a blue background, and the file name will be shown below the list on the left-hand side. See File Editing for more details.

Extras

Extra functionality can be found when the user right-clicks on a file record. A context menu will appear, allowing the user to either view the file in Notepad (and perhaps edit it there) or (Careful With That Axe, Eugene!) delete it. Note that deleting the file need not be as catastrophic as it sounds, WinFreeREG takes care to ensure that the delted file can be recovered from the Recycle Bin.
